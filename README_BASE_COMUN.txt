BASE COMUN - TPI FOOD STORE
===========================

Esta carpeta contiene la base compartida del proyecto.

Incluye:

- TPIProgra2.java
- MenuPrincipal.java
- Base.java
- Calculable.java
- Enums: Rol, Estado, FormaPago
- Excepciones comunes

Importante:
La base comun sola puede no compilar hasta que se suban las 4 partes, porque MenuPrincipal referencia Categorias, Productos, Usuarios y Pedidos.

Orden recomendado:
1. Base comun
2. Parte 1 - Categorias
3. Parte 2 - Productos
4. Parte 3 - Usuarios
5. Parte 4 - Pedidos
